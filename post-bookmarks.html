بوکمارک‌ها رو جدی بگیر

ما هر روز ده‌ها سایت رو باز می‌کنیم. اگه هر بار بخوای آدرسش رو تایپ کنی یا سرچ کنی، وقت زیادی هدر میره. بوکمارک‌ها اینجا به کمک میان.

تعریف ساده: بوکمارک مثل یه میانبره که با یه کلیک می‌بری به سایت مورد نظرت.

اضافه کردن بوکمارک

توی جیبی اضافه کردن بوکمارک خیلی ساده‌ست:

۱

روی + کلیک کن

توی بخش بوکمارک‌ها، دکمه + رو بزن.

۲

اطلاعات رو وارد کن

اسم سایت، آدرس URL و یه آیکون انتخاب کن.

۳

ذخیره کن

روی ذخیره کلیک کن و تمام!

گروه‌بندی بوکمارک‌ها

به جای اینکه همه بوکمارک‌ها رو یه جا بریزی، اون‌ها رو گروه‌بندی کن:

  • کار: Gmail، Slack، Notion، GitHub
  • سرگرمی: یوتیوب، نتفلیکس، اسپاتیفای
  • خبر: توییتر، ردیت، خبرگزاری‌ها
  • یادگیری: یوتیوب آموزشی، Coursera، Udemy

آیکون اختصاصی

برای هر بوکمارک یه ایموجی انتخاب کن. اینطوری خیلی سریع‌تر پیداش می‌کنی:

  • 📧 برای ایمیل
  • 📺 برای یوتیوب
  • 💼 برای لینکدین
  • 🐙 برای گیت‌هاب
نکته: می‌تونی از آیکون‌های سفارشی هم استفاده کنی. فقط کد SVG رو paste کن.

مرتب‌سازی با Drag & Drop

بوکمارک‌ها رو با کشیدن و رها کردن (Drag & Drop) می‌تونی جابجا کنی. سایت‌هایی که بیشتر استفاده می‌کنی رو بالا بذار.

وارد کردن از مرورگر

اگه بوکمارک‌هایی توی کروم داری، می‌تونی اون‌ها رو به جیبی import کنی:

  1. Settings → Import/Export رو باز کن
  2. «Import from Chrome» رو بزن
  3. بوکمارک‌هایی که می‌خوای رو انتخاب کن

حالا دیگه یه حرفه‌ای شدی! بوکمارک‌هات مرتب و آماده‌ست.
